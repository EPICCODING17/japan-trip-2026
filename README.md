# Japan Trip 2026 LIFF Todo

Static LINE LIFF todo page for the Japan trip checklist.

## Files

- `index.html` - main LIFF page
- `line_todo_japan_trip_apps_script.gs` - Google Apps Script backend for the linked Google Sheet

## Deploy

Enable GitHub Pages from the repository settings:

1. Go to Settings > Pages.
2. Source: Deploy from a branch.
3. Branch: `main` and folder `/ (root)`.

Then use the GitHub Pages URL as the LIFF Endpoint URL.
